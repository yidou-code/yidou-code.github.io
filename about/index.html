# 关于本站

这是一个使用 VuePress Theme Vdoing 主题搭建的示例文档网站，主要用于展示该主题的基本功能和使用方法。

# 关于主题

VuePress Theme Vdoing 是一个基于 VuePress 的主题，由 Evan Xu (opens new window) 开发。该主题专注于文档和知识管理，支持多种布局方式和功能。
