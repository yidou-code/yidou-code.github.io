# 2025年3月学习总结

提示

本文记录了2025年3月的学习内容、成果与思考。

# 学习内容

本月主要专注于以下几个方面的学习：

  1. 前端技术

    • javascript的初级入门
    • 力扣的js的初级算法题
    • vue初步了解

  2. 英语四级

    • 单词和听力提升

  3. 程序设计

    • c++初步入门

# 学习成果

  • 力扣简单免费算法题练习完成

# 项目进展

  • 使用js制作一个简单的轮番图和计时器等小项目

# 技术难点突破

js的语法逻辑和箭头函数运用不熟练。理解有误差

# 下月计划

  • 着重学习英语（听力+作文）
  • 学校任务优先级提升
  • 学习vue2+vue3
  • 提高代码的敲写量
  • 加深js 和 c++ 的理解
  • 每天坚持读书30min

# 思考与反思

本月学习没有注重效率，学习预期与时间调理极不平衡； 个人受环境影响，自制力偏低. 下个月继续努力，加油！（多去图书馆）
